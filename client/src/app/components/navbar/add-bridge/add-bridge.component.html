IP Address Lutron App > Settings > Advanced > Integration > Network Settings Please enter a valid IP address. IP address is required.

Lutron App > Settings > Advanced > Integration > Telnet Support Port can't be less than 1. Port can't be higher than 65535. Port must be a whole number between 1 and 65535. Port is required.
Integration Report Lutron App > Settings > Advanced > Integration > Send Integration Report
